## 왜 ScriptPlayer+인가

ScriptPlayer+는 이미 로컬 미디어와 스크립트를 갖고 있지만, 여기저기 덧붙인 느낌이 아니라 지금 기준으로도 쓸 만한 플레이어를 원하는 사람을 위한 앱입니다.
핵심은 단순합니다. 더 깔끔한 재생 경험, 더 정리된 디바이스 제어, 덜 번거로운 라이브러리 워크플로우입니다.

## 주요 기능

### 재생과 라이브러리

- 로컬 영상 파일 재생: `MP4`, `MKV`, `AVI`, `WebM`, `MOV`, `WMV`
- 로컬 오디오 파일 재생: `MP3`, `WAV`, `FLAC`, `M4A`, `AAC`, `OGG`, `OPUS`, `WMA`
- 시각적 라이브러리 작업 공간과 접을 수 있는 기존 파일 패널 제공
- 번들 펀스크립트 자동 감지, 별도 스크립트 폴더 지원, 고유하게 매칭되는 폴백 스크립트 자동 선택
- 여러 스크립트 번들이 잡히는 미디어용 빠른 스크립트 변형 패널
- 외부 자막 파일 자동 감지 및 수동 자막 불러오기 지원
- 파일 리스트에서 호버 비디오 프리뷰 제공
- 경로, 파일명, 별점, 재생 시간, 수정 시간 기준 정렬 지원
- 현재 미디어 반복, `자동 다음 재생`, 랜덤 재생, 재생 속도 조절을 분리해서 제공
- 드래그 앤 드롭으로 미디어 바로 열기 지원
- 오디오 메타데이터와 작품 폴더를 함께 탐색하는 커버 아트·자막 감지
- 작은 음원을 위한 선택적 `150%` 볼륨 증폭

### 스크립트 시각화와 제어

- 여러 축을 동시에 선택하고 높이를 조절할 수 있는 실시간 스크롤 타임라인
- 속도 기반 색상 시각화를 적용한 전체 미디어 히트맵
- 재생바에서 바로 조절하는 빠른 `STR` 스트로크 컨트롤
- 미디어별 저장을 지원하는 Range Extender와 스트로크 반전 토글
- `-60초`부터 `+60초`까지 직접 소수 입력이 가능한 미디어별 스크립트 오프셋
- 스크립트가 없는 미디어용 랜덤 폴백 스트로크 생성
- 갭 채우기, 지연 가능한 자동 스킵, 희소 스크립트 보정
- `axes[]`, `channels{}` 통합 파일 및 축별 분리 파일의 다축 로딩·라우팅

### 디바이스와 스크립트 소스

- `The Handy` API v2 업로드·동기화, 네트워크 진단, 구형 하드웨어 호환 모드
- `Intiface / Buttplug` 선형, 회전, 스칼라 피처에 대한 다축 매핑 지원
- `FunOSR` 시리얼·Bluetooth COM 프로필, 축 매핑, 재연결, 스무딩 지원
- 전용 액세서리 페어링, 자동 재연결, 키 매핑, 플래셔 바로가기
- `JAVPlayer`, `JAVPlayerEZ`, `mpv`, `PotPlayer`, `VLC`, 사용자 지정 실행 파일용 실험적 외부 플레이어 모드
- Electron에서 직접 디코딩하지 못하는 영상을 위한 실험적 FFmpeg 호환 재생
- 인앱 `EroScripts` 로그인, 탐색, 검색, 다운로드 지원
- 로컬 머신 기준 EroScripts 로그인 세션 유지

## v0.3.0 주요 변경점

- 통합 내비게이션과 시각적 라이브러리 작업 공간을 중심으로 UI를 개편하고 기존 파일 패널은 접어서 함께 쓸 수 있게 했습니다.
- `axes[]` / `channels{}` 통합 다축 스크립트, 여러 축 동시 타임라인, 타임라인 높이 조절을 추가·보강했습니다.
- Range Extender, 미디어별 출력 범위, 모션 스무딩, `±60초` 숫자 직접 입력 스크립트 오프셋을 추가했습니다.
- Handy HSSP 복구와 자동 재생 안정성을 개선하고 구형 Handy 호환 모드와 네트워크 지연 진단을 추가했습니다.
- FunOSR 시리얼·Bluetooth COM 프로필, 재연결 안전성, 일시정지·홈 동작, 부드러운 출력 프리셋을 보강했습니다.
- JAVPlayer 계열, mpv, PotPlayer, VLC, 사용자 지정 플레이어용 실험적 외부 플레이어 모드를 추가했습니다.
- Chromium에서 재생하지 못하는 영상을 위한 실험적 FFmpeg 호환 엔진을 추가했습니다.
- 오디오 커버·자막 탐색과 시작 성능을 개선하고 `150%` 볼륨 증폭 기능을 추가했습니다.
- 전용 액세서리 자동 재연결, 페어링, 플래셔 진입 흐름을 개선했습니다.
- 동일 버전 핫픽스 감지, 변경 내용 표시, 인앱 설치 연결을 추가했습니다.
- 플레이리스트 스크립트 재매칭, 스크립트 폴더 인덱싱, 갭 채우기, 검은 화면 경고 오탐 문제를 수정했습니다.

## 현재 참고 사항

- 영상 컨테이너 내부의 내장 자막 트랙은 아직 파싱하지 않습니다. 현재는 외부 자막 파일을 사용해야 합니다.
- 외부 플레이어 모드는 모든 지원 대상을 실행할 수 있지만 재생 제어와 동기화 수준은 플레이어마다 다릅니다.
- FFmpeg 호환 재생은 실험 기능이며 FFmpeg 실행 파일이 설치되어 있거나 설정에서 지정되어야 합니다.
- 장치 출력은 현재 한 번에 하나의 공급자만 사용하며 Handy와 Intiface / FunOSR 동시 출력은 아직 지원하지 않습니다.

## 소스 공개 범위

ScriptPlayer+는 현재 EULA에 따라 패키징된 데스크톱 바이너리로 배포됩니다. 공개 저장소는 릴리스, 업데이트 메타데이터, 문서, 이슈 추적에 사용하며 전체 애플리케이션 소스는 공개 배포하지 않습니다.

## 키보드 단축키

| 키 | 동작 |
| --- | --- |
| `Space` / `K` | 재생 / 일시정지 |
| `Left` / `Right` | `-5초 / +5초` 이동 |
| `Shift + Left / Right` | `-10초 / +10초` 이동 |
| `Up` / `Down` | 볼륨 `+5% / -5%` |
| `F` | 전체화면 전환 |
| `M` | 음소거 전환 |
| `Ctrl + ,` | 설정 열기 |
